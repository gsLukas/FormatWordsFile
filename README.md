# FormatWordFile


Version="0.1"

Basicamente, esse script vai formatar um wordlist/dict.
 Ex:
      palavra1
      palavra2
      palavra3
      palavra4
      teste1
      teste2
      teste3
      teste4

Removendo todas palavras com quantidade igual ou menor que 8 caracters.
Removendo todas palavras com quantidade igual ou maior que 8 caracters.
